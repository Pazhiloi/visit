Технологии

HTML5

Язык для структурирования и представления содержимого всемирной паутины.

CSS3

Формальный язык описания внешнего вида веб-сайта,написанного с использованием HTML

JavaScript

Язык программирования, который добавляет интерактивность на ваш сайт (например: игры, отклик при нажатии, динамические стили, анимация).

Jquery

Популярная библиотека JavaScript.Создана с целью облегчить разработчикам использование JavaScript на веб-сайтах.

Flexbox

Flexbox предоставляет инструменты для быстрого создания сложных, гибких макетов, и функции, которые были сложны в традиционных методах CSS.

Grid

Вводит двухмерную сетку к CSS. Сетки могут быть использованы для выделения основных областей страницы либо небольших элементов интерфейса пользователя.

Animation

Это библиотека, в которую встроены десятки классных кроссбраузерных анимаций, которыми можно пользоваться при разработке своего проекта.

BEM

Компонентный подход к веб-разработке.Он позволяет легко и быстро разрабатывать интерфейсы любой сложности и повторно использовать существующий код

Git

Это набор консольных утилит, которые отслеживают и фиксируют изменения в файлах

Gulp

это инструмент для автоматизации рутинных задач, которые возникают при веб-разработке.

SCSS

Препроцессор, который ускоряет работу

SVG

Язык разметки масштабируемой векторной графики,предназначен для описания двумерной векторной и смешанной векторно/растровой графики в формате XML.
